# Simple Calculator

A basic web-based calculator built using HTML, CSS, and JavaScript. This calculator allows users to perform simple arithmetic operations like addition, subtraction, multiplication, and division.

## Features

- Perform basic mathematical operations:
  - Addition (`+`)
  - Subtraction (`-`)
  - Multiplication (`*`)
  - Division (`/`)
  - Modulus (`%`)
- Clear input (`C` button)
- Backspace functionality (`←` button)
- Supports decimal inputs

## Demo

You can try out the Simple Calculator by cloning this repository and opening the `Simple_Calculator.html` file in your web browser.

## Installation and Usage

1. **Clone the repository**:
    ```bash
    git clone https://github.com/Kushal-Narendracumar7/Simple-Calculator.git
    ```
2. **Navigate to the project folder**:
    ```bash
    cd Simple-Calculator
    ```
3. **Open the `Simple_Calculator.html` file** in your browser:
    ```bash
    open Simple_Calculator.html
    ```
    or simply double-click on the `Simple_Calculator.html` file in your file explorer.

## Files

- **index.html**: The main HTML file that contains the structure of the calculator.
- **style.css**: The CSS file for styling the calculator interface.
- **script.js**: The JavaScript file that contains the functionality and logic for the calculator operations.

## How to Use

1. Input your expression using the number buttons (`0-9`) and operator buttons (`+`, `-`, `*`, `/`, `%`).
2. Press the `=` button to evaluate the expression and get the result.
3. Use the `C` button to clear the entire input, or the `←` button to remove the last character.
4. You can use parentheses (`(` and `)`) to group expressions for complex calculations.


## Technologies Used
